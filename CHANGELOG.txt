[v25.12.01] 2025-12-01
----------------------------------------------
🔧 개선 사항
- Shift 반복토글 속도 개선
  · 시퀀스 버튼 반복 속도가 다운홀드와 동일해짐
  · 반복 중 간헐적 끊김 현상 해결
- 버튼 라벨 색상으로 실행 상태 구분
  · 밝은 청록색: 실행 중
  · 진한 청록색: 반복 대기/쿨타임
  · 흰색: 일반 상태
- 단축창 핫키 기본키로 설정시에도 휠클릭 사용 가능하도록 수정
  · F12 키에 스턴올리고, 단축창 핫키 설정에서 F12->F12로 지정, 휠클릭사용 체크시 F12키 입력하면 대상에 바로 스턴 시전
- Shift 반복토글 로직 개선
  · Shift + 핫키,버튼 으로 반복중 다른 핫키를 입력할 경우 반복중이던 시퀀스,단축키를 일시정지하고 입력한 핫키,버튼(시퀀스)를 우선 실행보장.
  · 예를들어, F5에 물약을 올리고 시프트+Q로 자동물약을 실행중인 경우, 변신시퀀스(버튼,핫키)를 추가 실행할 경우 변신시퀀스의 실행이 완전 보장됨.

[v25.11.15] 2025-11-15
----------------------------------------------
🆕 새로운 기능
- 키입력 홀드시간 설정 추가
  · 타이밍 설정에서 조절 가능 (기본값: 50ms)
  · 게임에 따라 최적화 가능 (권장: 10~100ms)

🔧 개선 사항
- 휠클릭 미사용 시 키보드 반복 속도 개선
  · 짧게 누르면 1번만 실행 (기존: 2번 실행되던 문제 해결)
  · 길게 누르면 자연스러운 반복 (초기 지연 500ms + 이후 30ms 간격)
- 키 전환 응답성 향상 (e→f 등 빠른 키 전환 시 즉시 반응)
- 타이밍 설정 사용자 정의 강화 (하드코딩 제거)
- 매크로 핫키에 등록되지 않은 키는 버튼 핫키로 지정 가능하도록 수정
- 123,qwer,asdf 키도 매크로 핫키에서 사용 안할 경우, 버튼 핫키로 지정 가능
**매크로핫키 사용 안하는 방법 : F1~12까지 매핑되는 키를 동일하게 설정 ex) F1 - F1 , F12 -F12
- 자동업데이트 로직 개선. 실행파일 실제 업데이트 간헐적으로 실패하는 문제 수정

🐛 버그 수정
- 설정창 최소화 시 자동 복원 기능 추가
- 오타 수정 및 중복 코드 제거
