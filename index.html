💜 이레캔 애드온 이야기

이레캔 애드온은 이레이즈 매직 + 캔슬레이션 마법을 빠르게 연달아 사용해서 상대방을 무력화 시키던 그시절 컨트롤과 향수를 떠올리며, 이제는 아재가 되어버린 유저분들을 위하여 제작하였습니다.
이레캔 애드온과 함께 훨씬 더 편안하고 즐겁게 게임하세요!

💡 결론

이레캔 애드온은 게임을 편리하게 만드는 도구입니다.
게임을 해킹하거나 조작하는 것이 아닌,
키보드/마우스 입력을 자동화하는 편의성 프로그램입니다.

🛡️ 게임 운영진 여러분께:
본 프로그램은 게임 클라이언트를 전혀 수정하지 않으며,
단순히 사용자의 키보드/마우스 입력을 자동화하는 매크로입니다.
게임의 공정성을 해치지 않는 편의성 도구로 이해해주시기 바랍니다.

😓 20년 전 인터페이스, 불편하셨죠?

세계최초 리니지 클래식 애드온 - 익숙하지만 불편했던 경험들

이런 불편함, 공감하시나요?

  • 휠스크롤 굴리고 휠버튼 클릭해야 스킬 시전, 원하는 단축키 위치로 빠르게 전환 어려움
  • F1 F2 F3 단축창 스왑하며 F12는 손도 안가고
  • 사용할 스킬·아이템 많으면 긴급한 순간 손꼬임
  • F1~F12까지 거리 멀어서 사용하기 어려움
  • 반복되는 마우스 클릭, 단축키 입력 피로도 증가

💡 이제 롤/와우처럼 QWER로 편하게 플레이하세요! 🎯

  • QWER 스킬 난사: 핫키 한 번으로 바로 대상에게 스킬 시전, 클릭 한 번을 줄여주는 편리함
  • 무빙 콤보: 단축창 F5~F12 스킬을 휠클릭처럼 사용, 무빙트리플·무빙선버 간편하게
  • 핫키 커스텀: 휠클릭 방식 적용/미적용 선택 가능, 클래스/상황별 프리셋 3개 전환

2️⃣ 버튼 UI (애드온) 기능

게임 화면에 직접 표시되는 커스텀 버튼 오버레이

  • 최대 20개 버튼: 원하는 만큼 커스텀 가능
  • 드래그 이동: 게임 화면 원하는 곳에 자유 배치
  • 투명도 조절: 0~255단계 (시야 방해 없음)
  • 가로/세로 전환: 레이아웃 자유롭게 변경
  • 1~3행 배치: 공간 효율적으로 활용

3️⃣ 시퀀스 시스템

복잡한 작업을 자동화하여 편리하게

🔁 스킬 자동 반복

  • 20개 버튼에 자유롭게 지정
  • 버튼만 딸깍하면 자동 유지
  • 잠수 중 군버프, 쿨타임 버프
  • 반복 간격 자유 설정

⚡ 쉽고 빠른 타이핑

  • 서버 명령어: .빨 .주 .말 .칼렉풀기 .자동칼질 자주 쓰는 명령어를 빠르게
  • 귀찮은 타이핑 간편하게

✨ 버프 한방에

  • 요정버프, 다엘버프, 기타 버프 등
  • 버튼 클릭 한 번으로 해결
  • 복잡한 버프 시퀀스를 클릭 한 번에

🏥 자힐/자뮨

  • 셀프 힐, 셀프 이뮨
  • 단축키/버튼 한 번으로 내 캐릭터에 시전
  • 클릭 두 번이 한 번으로

손 많이 탄다는 법사, 요정도 신컨 가능 • 귀찮은 반복도 편하게 • 누구나 쉽게 사용
